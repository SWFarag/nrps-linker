NRPS-Linker Workflow:

  1. Download all known NRP biosynthetic gene clusters from the MIBiG repository.

  2. Download the whole NCBI RefSeq Bacterial genomes.

  3. Each database was scanned for NRP gene clusters using the AntiSMASH tool, that allows rapid genome-wide identification, annotation and analysis of secondary metabolite biosynthesis gene clusters in bacterial and fungal genomes.

  4. Run NRPS-Linker on all identified NRP gene clusters, extract all possible IMLs and update our IMLs database


BGC Parser

Aim: BGC Parser is an algorithm that extracts IMLs from NRP biosynthetic gene clusters.

Input: The user should a least upload one NRP biosynthetic gene cluster in Genbank format, please check example of Uploaded files.

Output:

  1. A table of all extracted linkers. An IML will be presented in the following format A1-Linker-A2:

    • A1: The adenylation substrate of module one.
    • Linker: The inter-modular linker between two successive modules
    • A2: The adenylation substrate of module two

  2. An extraction summary: which point out to how many files got processed, how many linkers were retrieved and how many clusters lack IMLs


IMLs Database

IMLs database consist of two Libraries:

  1. MIBiG IMLs library: which contains more than 880 inter-modular linkers retrieved from 408 non-ribosomal peptide BGCs.
  2. NCBI IMLs library: which contains more than 37600 inter-modular linkers retrieved from 51810 non-ribosomal peptide BGCs.
Both libraries share the following columns:
  • A1: The adenylation substrate of module one.
  • Linker: The inter-modular linker between two successive modules
  • A2: The adenylation substrate of module two
  • Length: The length of the linker
  • Organism_name: The name of the organism the BGC belongs to.
Both libraries are searchable and the data can be sorted in multiple ways. Moreover, if user are interested in a particular section of the table, they get to export that section as a CSV file.


Downloads

Users get to download the entire database locally on their machines.
